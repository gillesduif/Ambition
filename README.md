# Holonoid Backend
Een Flask-gebaseerde backend voor Holonoid. Ondersteunt uploaden, statuscontrole en meer.

## Structuur
- `app.py`: Hoofdapplicatie.
- `routes`: Bevat de verschillende API-routes.
- `utils`: Helperfuncties en logging.
